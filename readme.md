##

### Bot para Facebook version 0.1

#### Como usar
- Possuir Node JS instalado em sua máquina.

```bash
> git clone https://github.com/Quirin0/facebook-bot
```

#### Dependências
```bash
> npm install puppeteer
```

#### Execução
```bash
> node app.js
```
